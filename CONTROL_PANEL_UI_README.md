# 🚀 LLM项目控制台 - Web UI版

一个美观、现代化的Web控制面板，用于管理本地LLM项目的所有系统和服务。

## ✨ 特性

- **🎨 现代化UI设计** - 渐变色彩、毛玻璃效果、流畅动画
- **🚀 一键式操作** - 启动/停止系统、批量管理
- **📊 实时状态显示** - 服务状态指示灯、动态更新
- **🛠️ 多系统集成** - AI监理、Claude Code、记忆系统等
- **📱 响应式设计** - 适配各种屏幕尺寸

## 🖼️ 界面预览

控制台包含以下系统模块：

1. **🤖 AI监理系统** - 智能代码审查和自动修复
2. **🔧 Claude Code** - AI代码生成和项目脚手架  
3. **🧠 记忆系统** - 向量数据库和知识管理
4. **⚡ 性能加速** - Go/Rust高性能引擎
5. **🔊 声音系统** - 文本转语音服务
6. **📊 监控系统** - 实时监控和日志管理

## 🚀 快速启动

### 方式1：简洁版（推荐）
直接在浏览器中打开，无需服务器：

```bash
# 方法1：使用启动脚本
chmod +x launch_control.sh
./launch_control.sh
# 选择 1

# 方法2：直接打开HTML
open control_panel_simple.html
```

### 方式2：完整版
带后端API支持的完整版本：

```bash
# 启动Web服务器
chmod +x start_control_panel.sh
./start_control_panel.sh

# 浏览器会自动打开 http://localhost:8888
```

### 方式3：命令行版
传统的终端控制中心：

```bash
./control_center.sh
```

## 📁 文件说明

```
control_panel_simple.html   # 简洁版UI（独立运行）
control_panel.html         # 完整版UI（需要服务器）
control_panel_server.py    # Python后端服务器
start_control_panel.sh     # 完整版启动脚本
launch_control.sh          # 统一启动器
```

## 🎮 使用指南

### 系统控制
- 点击每个系统卡片的"启动系统"按钮来启动服务
- 状态指示灯会显示运行状态（绿色=运行中）
- 系统启动后按钮会变为"停止系统"

### 快速操作
- **全部启动** - 一键启动所有系统
- **全部停止** - 一键停止所有系统
- **创建项目** - 快速创建新项目
- **打开终端** - 在项目目录打开终端
- **查看日志** - 查看系统日志
- **健康检查** - 检查所有服务状态

### 键盘快捷键
- `ESC` - 关闭弹出窗口

## 🔧 自定义配置

### 修改系统配置
编辑 `control_panel_server.py` 中的 `SERVICES` 字典来自定义系统配置。

### 修改UI样式
编辑 HTML 文件中的 CSS 样式部分来自定义界面外观。

## 🐛 故障排除

### 简洁版无法控制服务？
简洁版HTML是一个演示界面，实际控制服务需要使用完整版。

### 端口被占用？
```bash
# 检查8888端口
lsof -i :8888

# 终止占用进程
kill -9 <PID>
```

### Flask未安装？
```bash
pip3 install flask flask-cors
```
